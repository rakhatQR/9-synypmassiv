Бір өлшемді массивтер

Пән мұғалімі: Еламанов Рахат

Сабақтың мақсаты:

Python тілінде кездейсоқ сандардан тұратын бір өлшемді массив құруды және массив элементтерімен негізгі амалдарды (іздеу, қосу, сүзу) орындауды үйрену.

Массивті құру үлгісі: 
import random
A = [random.randint(0, 50) for i in range(10)]
print("Массив:", A)

1-тапсырма: Максимум және Минимум

Оңай

Шарты:

Массивтегі ең үлкен (максимум) және ең кіші (минимум) сандарды табыңыз және экранға шығарыңыз.

2-тапсырма: Қосынды мен Орташа мән

Орташа

Шарты:

Массивтегі барлық сандардың қосындысын және олардың арифметикалық ортасын (орташа мәнін) есептеп шығарыңыз.

Кеңес: sum() және len() функцияларын қолданыңыз.

3-тапсырма: Тақ сандарды сүзу

Қызықты

Тапсырма шарты:

Массив ішіндегі тек тақ сандарды теріп алып, оларды экранға шығарыңыз және олардың жалпы санын көрсетіңіз.

Мысал: Жұп сандарды табу 
import random

A = [random.randint(0, 50) for i in range(10)]
print("Бастапқы массив:", A)

zhup_sandar = [] # Жұп сандарды жинау үшін бос тізім
count = 0        # Санауыш

for x in A:
    if x % 2 == 0:        # Егер сан 2-ге қалдықсыз бөлінсе
        zhup_sandar.append(x)
        count += 1

print("Жұп сандар:", zhup_sandar)
print("Жұп сандардың саны:", count)
Назар аударыңыз: Тапсырмада сіз тақ сандарды табуыңыз керек (қалдық 0-ге тең емес).
